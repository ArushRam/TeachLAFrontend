## This is the frontend code for the <Teach LA> Helper site

# Dependencies
1. yarn (or npm)

# To Get Started
1. Clone the repo onto your machine
2. cd into the project directory
3. yarn (or npm) install
4. yarn (or npm) start
5. The Client should be viewable on localhost:8080
